4. Cloud Account Access

Permissions Required

To provide our service, CostSage requires you to grant us access to your cloud accounts:

AWS Access

  • Read-only analysis: We require an IAM role with read-only permissions to EC2, RDS, S3, Lambda, CloudWatch, and other AWS services.
  • Execution permissions (optional): If you approve specific changes, we require minimal-scope execution permissions (e.g., EC2 resize, S3 lifecycle rules) scoped to specific resource tags or regions.
  • All execution permissions are logged in CloudTrail for full audit visibility.

Azure Access

  • Read-only analysis: We require a Service Principal with Reader role on subscriptions for analysis.

  • All execution is logged in Azure Activity Log.
